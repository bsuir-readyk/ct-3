# Task 2 - вариант 10

Для заданного вариантом способа преобразование текстовых данных разработать:
•	flex-описание анализатора;
•	программное средство, сгенерированное на основе описания и выполняющее заданное преобразование.
Программное средство должно преобразовывать данные, поступающие из стандартного потока ввода stdin и выводить результат своей работы в стандартный поток вывода stdout.

Текстовый документ
Зашифровать текст шифром Цезаря. Шифрованию подвергать только латинские буквы.

## Как собрать

Выполнить в каталоге проекта:

```bash
cd /Users/readyyyk/prog/occpro/monorepo/src/projects/uni/kt/2
make main
```

Команда `make main`:
- запускает `flex` для генерации `lex.yy.c` из `task2.l`;
- компилирует `main.c` и `lex.yy.c` в исполняемый файл `main`.

## Как использовать

Программа читает данные из стандартного ввода (`stdin`) и печатает результат в стандартный вывод (`stdout`).

Примеры:

- Зашифровать введённую строку:

```bash
echo "Hello, World!" | ./main
```

- Зашифровать содержимое файла `input.txt` и записать результат в `output.txt`:

```bash
./main < input.txt > output.txt
```

В обоих случаях:
- все латинские буквы шифруются шифром Цезаря со сдвигом 3 (A→D, X→A, a→d, z→c и т.п.);
- все остальные символы (пробелы, цифры, знаки препинания, кириллица и пр.) остаются без изменений.
